# MNIST Neural Network
This application classifies digits from 0-9 by using a simple neural network that is made from scratch by only using NumPy.

## Technologies Used
- FastAPI
- NumPy
- SvelteKit
- Chart.js
- Nginx
- Docker
- DigitalOcean
- GoDaddy
